# GitHub Dark Theme 🎨

GitHub Dark Theme for PowerShell ISE.  
It supports **Scripting Pane, Console Pane,** and **XML!**

## How to Install 📁

Open the ISE and go to
```
Tools -> Options -> Manage Themes -> Import
```
Enjoy the Theme 💙
